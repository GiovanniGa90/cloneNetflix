### Netflix Clone 🎬

Un clone dell'interfaccia di Netflix focalizzato sulla riproduzione fedele della UI/UX della nota piattaforma di streaming. Questo progetto mette in mostra le competenze nell'architettura a componenti e nella gestione di layout complessi e responsive.

* * *

### 🛠️ Tecnologie Utilizzate

*   **Frontend**: Angular / TypeScript
*   **Stile**: CSS3 / SASS / Bootstrap (Layout responsive per desktop e mobile)
*   **Dati**: Integrazione con API per il recupero dinamico dei contenuti (film, serie TV, locandine)

* * *

### 💻 Funzionalità Principali

*   **Dashboard Principale**: Slider e caroselli di film divisi per categorie.
*   **Interfaccia Responsive**: Ottimizzazione completa per la navigazione da smartphone, tablet e PC.
*   **Dettaglio Contenuto**: Modale o pagina dedicata per visualizzare le info del film selezionato.

* * *

### 🔧 Come avviare il progetto in locale

1.  Clona il repository: `git clone https://github.com`
2.  Installa le dipendenze: `npm install`
3.  Avvia l'applicazione: `ng serve`
4.  Apri l'app su `http://localhost:4200`
